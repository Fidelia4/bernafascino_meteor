# Berna Fascino - Site Vitrine

## Description
Site vitrine pour la maison Berna Fascino, réalisé avec **Meteor + React + Tailwind**.  
Permet de présenter l'histoire, la collection, les services et avis clients.

## Structure du projet
- `/imports/ui/pages` : pages React (Accueil, A propos, Contact, etc.)
- `/public/images` : images du site
- `/public/videos` : vidéos du site
- `/imports/startup` : configuration Meteor

## Prérequis pour l'installation
- Node.js >= 20
- Meteor >= 2.9
- Git
- Navigateur moderne (Chrome, Edge, Firefox)
- Connexion Internet

## Installation locale
```bash
# Cloner le projet
git clone https://github.com/ton-utilisateur/berna-fascino.git
cd berna-fascino

# Installer Meteor
curl https://install.meteor.com/ | sh

# Installer les dépendances
meteor npm install

# Lancer le projet
meteor
